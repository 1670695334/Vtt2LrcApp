# VTT转LRC 安卓工具

一个简单的 Android App：选择包含 `.vtt` 字幕的文件夹，递归转换所有 `.vtt`，并在原目录生成同名 `.lrc` 文件。

## 功能

- 批量转换文件夹内所有 `.vtt`
- 支持子文件夹递归转换
- 输出同名 `.lrc`
- UTF-8 BOM 编码
- LF 换行
- 适合网易云音乐识别滚动歌词

## 手机上怎么用

1. 安装 APK。
2. 打开 App。
3. 点击“选择文件夹并转换”。
4. 选择含 `.vtt` 的音乐/字幕文件夹。
5. 授权后自动转换。

## GitHub 自动打包 APK

把整个项目上传到 GitHub 后：

1. 打开仓库页面。
2. 点顶部 `Actions`。
3. 选择 `Build Android APK`。
4. 点 `Run workflow`。
5. 等构建完成后，进入本次运行结果。
6. 在 `Artifacts` 下载 `Vtt2Lrc-debug-apk`。
7. 解压后得到 `app-debug.apk`。

## 注意

- 这是 debug APK，手机安装时可能提示“未知来源”，允许安装即可。
- 安卓 11+ 对目录写入有限制，所以本 App 使用系统文件夹选择器授权，不需要申请存储权限。
